2. What Switchman is

Switchman is a CLI tool and coordination layer for parallel AI coding agents. It provides file locking, task queues, lease management, and merge coordination for software development workflows.

Switchman is offered in two tiers:

  • Free tier — open source, MIT licensed, no account required, up to 3 concurrent agents
  • Pro tier — paid subscription, requires a GitHub account, provides unlimited agents and additional features
