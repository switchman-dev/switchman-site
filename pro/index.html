FAQ

Common questions

Does the free tier still work exactly as before?
Yes, completely unchanged. The MIT core — full CLI, MCP server, file locking, task queues, lease management — stays free forever. Pro is entirely additive. Nothing disappears if your subscription ends.
What's the real difference between Free and Pro day to day?
Free hits the 3-agent cap and uses a local planning heuristic. Pro removes the cap and unlocks the full planning API — the intelligence behind switchman start that reads your repo, branch, and CLAUDE.md to propose a confident parallel task breakdown. Session summaries are also Pro-only, which is the feature most developers say makes Switchman feel indispensable.
How do notifications work and are they free?
Yes — Slack and desktop notifications are available on both Free and Pro. Run switchman notify --slack #channel to enable Slack, or switchman notify --desktop for OS notifications. You get pinged when an agent finishes, gets blocked, or when a session completes.
Do I need to watch the terminal while agents run?
No — that's the point. Start agents, go do something else entirely, and Switchman pings you when something finishes or needs attention. switchman status gives you a full snapshot whenever you want to check in manually.
Does Pro require a constant internet connection?
No. The licence check caches locally for 24 hours. If you're offline, Switchman uses cached credentials for up to 7 days. Local coordination always works offline — cloud sync and notifications require connectivity.
Will this affect my existing git workflow?
No. Switchman doesn't change how git works and doesn't touch your branches outside its own worktrees. Removing it is as simple as deleting the .switchman folder. Your repo, branches, and history are completely unaffected.
Can I cancel anytime?
Yes. Cancel through Polar.sh — your subscription runs until the end of the billing period then stops. No fees, no questions. Your local SQLite data is always yours.
Is this safe to use on a company codebase?
Yes. Switchman only stores coordination metadata in the cloud — task titles, file names, lease states. It never reads or stores your actual code. All file operations happen locally. Recio Ltd is GDPR compliant, data stored in EU region.
What about team features?
Team is coming soon — shared task queue across teammates, global file claims across machines, and a 90-day audit trail. Join the waitlist to be first when it ships.
