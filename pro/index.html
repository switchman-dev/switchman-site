FAQ

Common questions

Does the free tier still work exactly as before?
Yes, completely unchanged. The MIT core — full CLI, MCP server, file locking, task queues, lease management, and switchman merge — stays free forever. Pro is additive.
Will this break our existing git workflow?
No. Switchman doesn't change how git works and doesn't touch your branches outside its own worktrees. Removing it is as simple as deleting the .switchman folder. Your repo, branches, and history are completely unaffected.
What happens if I go over 3 agents without Pro?
The CLI shows a clear message with the upgrade command. Nothing breaks silently — you see exactly what the limit is and how to fix it. Run switchman upgrade to open this page.
Does Pro require a constant internet connection?
No. The licence check caches locally for 24 hours. If you're offline, Switchman uses the cached credentials for up to 7 days before requiring a reconnect. Cloud sync requires connectivity but local coordination still works offline.
How does team billing work?
Each seat is billed individually. The developer who installs Pro pays for their own seat. There is no admin console — teammates join via a terminal invite command and set up their own billing through the same flow.
Can I cancel anytime?
Yes. Cancel through Polar.sh — your subscription runs until the end of the billing period then stops. No fees, no questions. Your local SQLite data is always yours.
Is this safe to use on a company codebase?
Yes. Switchman only stores coordination metadata in the cloud — task titles, file names, lease states. It never reads or stores your actual code. All file operations happen locally. Recio Ltd is GDPR compliant, data stored in EU region.
What if I need more than email support?
For enterprise requirements — SSO, audit logs, RBAC, dedicated SLAs — get in touch at hello@switchman.dev.
